# Banana-Ripeness-Predictor-

An AI-powered computer vision and machine learning system that predicts banana ripeness stages (Green, Yellow, Brown, Mushy) and estimates remaining shelf life.
Built using TensorFlow, Keras, OpenCV, and FastAPI, the project integrates an end-to-end ML pipeline from image preprocessing and model training to real-time cloud deployment on Heroku/Render.
